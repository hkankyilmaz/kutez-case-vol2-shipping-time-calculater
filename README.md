# Shipping Time Calculater

🔸 Shipping Time Calculater for hiring case<br>
🔸 This web app built with using [Vite](https://vitejs.dev/) + [React](https://reactjs.org/). <br>
🔸 Click and see live demo... [Here](https://kutez-case-vol2-shipping-time-calculater.vercel.app/).

## 💻 Setup <br>

### Install Dependencies

```
npm install
```

### Run the App in the Dev Server

```
npm run dev
```

### to Build for Production

```
npm build
```

## 🔥 Technologies and Libraries <br>

✔️ React.js <br>
✔️ dayjs <br>
✔️ react-icons <br>
